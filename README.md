# Philippine University Tracker 🎓

The **Philippine University Tracker** is a web-based platform designed to help students discover and explore higher education institutions across the Philippines. Developed as a collaborative project for the **ITIPTECH** subject, this tool simplifies the school-searching process by providing a centralized database of universities and colleges, categorized by region and province.

---

## 🚀 Features

* **Comprehensive Search:** Search through a massive dataset to find the institution that best fits your needs.
* **Location Filtering:** Integrated filters allow users to browse schools based on specific **Regions** and **Provinces**.
* **Interactive Maps:** Built-in **Google Maps** integration to visualize school locations geographically.
* **Direct Website Access:** A dedicated feature that directs users to the official website of their chosen university.
* **Contact System:** A professional contact form powered by **EmailJS**, allowing users to send inquiries or feedback directly to the team.

---

## 🛠️ Technical Stack & APIs

This project leverages modern technologies to provide real-time, accurate data:

| Technology/API | Purpose |
| :--- | :--- |
| **Wikipedia API** | Fetches comprehensive data regarding various universities and colleges. |
| **PSGC API** | Uses the *Philippine Standard Geographic Code* to categorize schools by location. |
| **LangSearch API** | Powers redirection logic to connect users with university web resources. |
| **Google Maps Platform** | Provides the mapping interface for location visualization. |
| **EmailJS** | Handles "Contact Us" messaging without requiring a backend mail server. |

---

## 📁 Project Structure

The application consists of several key pages:
* **Home Page:** Featuring a clean search interface for quick access.
* **Find University:** A dedicated portal with advanced filtering (e.g., Zamboanga Peninsula, Zamboanga del Norte).
* **About Us:** Information regarding the project's purpose and the development team.
* **Contact Us:** A professional communication portal for user support.
